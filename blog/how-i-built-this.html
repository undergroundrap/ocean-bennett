Engineering a 100/100 Portfolio

April 25, 2026·4 min read

Achieving a perfect Lighthouse score isn't just about deleting code—it's about precision engineering. On this project, we focused on eliminating Total Blocking Time (TBT) by deferring heavy JS execution until the main thread was idle.

The Strategy

Most developers let their animations run immediately. We used requestIdleCallback with a strict timeout to ensure the browser has already painted the critical content before the Starfield animation begins.

// Pure optimization
window.addEventListener("load", () => {
  requestIdleCallback(() => {
    initStarfield();
  }, { timeout: 2000 });
});

By inlining critical CSS and using vanilla HTML, we reduced the time-to-first-paint to under 0.8s on a simulated mobile device.
