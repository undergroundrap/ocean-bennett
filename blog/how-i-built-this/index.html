Engineering a 100/100 Portfolio

April 25, 2026·4 min read

In the modern web, performance is often treated as a secondary metric to be "fixed" after launch. For this portfolio, performance was the foundational constraint. The goal: a "Full House" Lighthouse score (100/100 across the board) while maintaining high-fidelity visuals like a GPU-accelerated starfield and 3D hero interactions.

1. Main Thread Shielding

The biggest threat to a 100 Performance score is Total Blocking Time (TBT). Heavy animations, even when GPU-accelerated, can jam the main thread during the critical audit window. We solved this by implementing a shielding strategy using requestIdleCallback.

// Deferring non-critical visuals
window.addEventListener("load", () => {
  const init = () => {
    renderStars();
    startHeroEngine();
  };

  if (window.requestIdleCallback) {
    requestIdleCallback(init, { timeout: 2000 });
  } else {
    setTimeout(init, 200);
  }
});

By delaying the Starfield initialization until the browser is idle, we reduced TBT to exactly 0ms, ensuring the First Contentful Paint (FCP) is never delayed by decorative logic.

2. Enterprise-Grade Security Hardening

Security headers aren't just for compliance; they are a statement of technical maturity. We implemented a strict Content Security Policy (CSP) and optimized for HSTS Preloading. By setting a 2-year max-age with the preload flag, this domain is hard-coded into browsers as HTTPS-only, eliminating the latency of initial redirects and mitigating MITM attacks.

3. Zero-JS Micro-Interactions

To keep the bundle size near zero, we moved UI depth into the CSS layer. The "Traits" tooltips and the personality pop-up on my name use ::after pseudo-elements and the content: attr(data-tooltip) property. This allows for rich, glassmorphism-styled hover effects with zero impact on the JavaScript execution budget.

4. Semantic Discovery for AI Agents

As we move toward an agentic web, technical SEO must evolve. We implemented JSON-LD Person Schema to explicitly link this domain to my professional entities (GitHub, LinkedIn) and provided a dedicated /llms.txt for AI crawlers to ingest my technical stack with high density and accuracy.
